# User Feedback — Collection Guide & Template

Level 4 requires **basic user feedback collection** alongside 10+ real wallet
interactions. This guide gives you a lightweight, honest way to gather it while
people test the live demo.

## How to collect

1. Share the live demo: https://polaris-stellar.vercel.app
2. Ask each tester to complete the onboarding flow (connect → fund → predict).
3. Send them the 6-question survey below (Google Form / Typeform / Excel).
4. Record their **wallet address** or a **tx hash** next to their response so the
   feedback and the on-chain proof line up for the submission.

## Survey questions

1. How easy was it to connect your wallet and fund a testnet account? (1–5)
2. How clear was the market / trading interface? (1–5)
3. Did you successfully place a prediction? (Yes / No — if no, what blocked you?)
4. How would you rate the mobile experience? (1–5 / N/A)
5. What is the single most confusing or frustrating part?
6. What one feature would make you use this again?
